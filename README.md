# Mayank Rathi - Portfolio Website

## Project Overview
A modern, responsive portfolio website showcasing Java/Spring Boot development skills, projects, and professional experience. The website will be deployed on Vercel/Netlify.

## Tech Stack
- Frontend: React.js with TypeScript
- Styling: Tailwind CSS
- Deployment: Vercel/Netlify
- Version Control: Git
- CI/CD: GitHub Actions

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Mayank-rathi/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm start
```

4. Build for production:
```bash
npm run build
```
